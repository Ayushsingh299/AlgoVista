# Scratch Paper

Visualize your own code here!

## Learning About Tracers

The project [Algorithm Visualizer](https://github.com/AlgoVista) has a visualization library in each
supported language ([JavaScript](https://github.com/AlgoVista/tracers.js)
, [C++](https://github.com/AlgoVista/tracers.cpp),
and [Java](https://github.com/AlgoVista/tracers.java)) to visualize codes.

There are five tracers in the library to visualize different types of data:

- [Array1DTracer](https://github.com/AlgoVista/AlgoVista/wiki/Array1DTracer)
- [Array2DTracer](https://github.com/AlgoVista/AlgoVista/wiki/Array2DTracer)
- [ChartTracer](https://github.com/AlgoVista/AlgoVista/wiki/ChartTracer)
- [GraphTracer](https://github.com/AlgoVista/AlgoVista/wiki/GraphTracer)
- [LogTracer](https://github.com/AlgoVista/AlgoVista/wiki/LogTracer)

There are also randomizers to help you create random data.

Check out the [API reference](https://github.com/AlgoVista/AlgoVista/wiki) for more information.

## Making Your Visualization Public

If you think other people would find your visualization useful, you can add it to the side menu
by [contributing to `AlgoVista/algorithms`](https://github.com/AlgoVista/algorithms/blob/master/CONTRIBUTING.md)
.
